mtou
=============

Multicast-unicast forwarding tool.

Requirements
=============

This project uses CMake.

This has been lightly tested on Mac, heavily used on Linux.

Building
=============

In the project top-level folder:

    mkdir build
    cd build
    cmake ..
    make
    cd ..

The binary is then in ./build/mtou.
